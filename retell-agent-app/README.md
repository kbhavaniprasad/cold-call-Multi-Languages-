# Retell AI Agent Interface - MERN Stack

A full-stack web application for interacting with Retell AI voice agents. Built with MongoDB, Express, React, and Node.js.

## Architecture

```
retell-agent-app/
├── backend/
│   ├── models/
│   │   └── Call.js
│   ├── routes/
│   │   ├── agent.js
│   │   └── calls.js
│   ├── server.js
│   ├── package.json
│   └── .env.example
├── frontend/
│   ├── public/
│   │   └── index.html
│   ├── src/
│   │   ├── components/
│   │   │   ├── AgentInterface.js
│   │   │   ├── CallHistory.js
│   │   │   ├── CallAnalytics.js
│   │   │   └── component.css
│   │   ├── App.js
│   │   ├── index.js
│   │   └── index.css
│   ├── package.json
│   └── .env.example
└── README.md
```

## Setup

Follow the `backend` and `frontend` setup instructions in their respective folders.
